# online_sales_analysis

# OPIS PROJEKTA
Ovaj projekat simulira sistem za prodaju, upravljanje proizvodima i korpom za kupovinu.

# KLASE I FUNKCIONALNOSTI
- **Product**: Modeluje proizvod sa cenom i količinom.
- **ProductManager**: Upravlja listom porizvoda (dodavanje i brisanje).
- **Cart**: Omogućava dodavanje artikala u korpu i izračunavanje ukupne vrednosti.
